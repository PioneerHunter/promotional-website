# 图片资源说明

## 需要添加的图片文件

### 个人资料部分
- `/public/avatar.jpg` - 个人头像图片（建议尺寸：256x256px，圆形显示）

### 社交媒体图标
在 `/public/social/` 目录下创建以下 SVG 图标：
- `facebook.svg` - Facebook 图标
- `twitter.svg` - Twitter/X 图标
- `github.svg` - GitHub 图标
- `youtube.svg` - YouTube 图标
- `linkedin.svg` - LinkedIn 图标

建议图标尺寸：24x24px，白色或浅色，适合在深色背景上显示。

### 产品图片
在 `/public/products/` 目录下添加产品图片：
- `product1.jpg` - 第一个产品图片
- `product2.jpg` - 第二个产品图片
- `product3.jpg` - 第三个产品图片

建议图片尺寸：1200x800px 或更大，用于卡片展示和放大预览。

## 快速开始

1. 创建目录结构：
   ```bash
   mkdir -p public/social public/products
   ```

2. 添加相应的图片文件到对应目录

3. 或者使用占位图片服务（如 placeholder.com）进行测试
